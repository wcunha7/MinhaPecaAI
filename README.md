# Minha Peça — Sprint 8 Investor Demo v0.8.0

Consolida as ideias das Sprints 4 a 8 em uma demonstração navegável.

## Executar
1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Abra `index.html`.
4. Clique em `Go Live`.
5. Acesse `http://localhost:5500/?v=080`.
6. Pressione `Ctrl + F5` se aparecer versão antiga.

## Incluído
Perfis, IA Vision simulada, busca por problema/código, diagnóstico, compatibilidade, guia, marketplace, score, orçamento, checkout, fornecedor, dashboard, histórico e visão para investidores.

## Importante
É uma demonstração funcional. IA, dados, fornecedores e pagamentos são simulados. Produção exigirá backend, banco, autenticação, integrações, segurança, LGPD e testes.
